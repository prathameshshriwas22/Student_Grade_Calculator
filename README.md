# Student_Grade_Calculator
"I developed a Student Grade Calculator using Python. The program takes marks as input and uses if-elif-else conditions to determine the grade. This project helped me understand decision-making logic in Python."
Student Grade Calculator

Description:
This project is a simple Python program that calculates a student's grade based on marks.

Features:
- Takes marks as input
- Calculates grade using conditional statements
- Displays grade result

Technologies Used:
Python
If-Else Conditions

How to Run:
1. Open grade_calculator.py
2. Run using Python
3. Enter marks when prompted
